URL Suggestion

  • To transliterate a webpage which is having unicode contents, give the url as it is.
    eg., http://te.wikipedia.org

  • If the site is using proprietary font and is supported by unicode conversion gateway, give the url as http://uni.medhas.org/unicode.php5?file=http://siteurl
    eg., http://uni.medhas.org/unicode.php5?file=http://www.gujaratsamachar.com

  • If the error "Please give full URL (or define a default IP to redirect to)" occurs, make sure that the url contains http:// . If the problem still exists, report it as a bug.
